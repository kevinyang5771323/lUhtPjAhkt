# 项目简介

## 核心功能
- 基于YOLOv8的螺栓缺陷检测，支持10类缺陷识别：鼓包、爆裂、圆形缺陷、裂纹、损伤、褶皱、划痕、工具痕迹、空洞、编码错误
- 支持多种输入方式：单张图片、图片文件夹、视频文件、本地摄像头
- 提供模型选择功能，支持自定义置信度和IOU阈值
- 实时显示检测结果：目标位置、目标总数、置信度等信息
- 支持批量检测，可查看每一张检测结果
- 支持检测结果保存功能
- 完整的数据集包含451张标注图片

## 适用场景
- 工业制造中的螺栓质量检测
- 产品生产线自动化质检
- 工业设备维护中的零部件检查
- 质量控制与研究分析

## 技术特点
- 采用YOLOv8深度学习模型进行缺陷检测
- 基于Python 3.9开发，使用OpenCV、PyQt5、Ultralytics、Torch1.9等技术栈
- 提供图形用户界面，操作简便
- 已完成模型训练，配置环境后可直接使用
